# Random-Streamlit-Weather-App

A minimal random Streamlit weather app that is ready to deploy from GitHub.

## Files to keep in the repository root

- `app.py` - the Streamlit entrypoint
- `requirements.txt` - Python dependencies for Streamlit Cloud

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy with Streamlit Community Cloud

1. Push this repository to GitHub.
2. In Streamlit Community Cloud, choose **New app**.
3. Select this GitHub repository.
4. Set the main file path to `app.py`.
5. Deploy.
